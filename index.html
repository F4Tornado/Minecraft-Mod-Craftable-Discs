Craftable Discs - Minecraft Mod

This mod allows you to craft music discs in Minecraft.

How to install:

  1. Install Forge 1.12.2
  2. Download the mod here
  3. Put it in the folder C:/Users/your-profile-name/AppData/Roaming/.minecraft/mods
  4. Run Forge in the Minecraft launcher, & hope it doesn't crash
